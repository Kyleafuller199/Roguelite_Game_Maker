## Overview
Roguelite Game Maker is a **web-based creation tool** designed to enable aspiring game creators to make, test, and share roguelite-style games without writing code. The tool will provide an intuitive visual editor where creators can define characters, cards, relics, and enemies, test encounters, and iterate quickly on balance and design.
